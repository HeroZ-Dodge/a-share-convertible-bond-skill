# A 股可转债配债规则

## ⚠️ 重要说明

**配债规则不是固定的！每只转债的每股配售额不同！**

每股配售额由发行公司根据发行规模和总股本确定，不同转债差异巨大:
- 金杨转债：8.55 元/股 → 1000 股可配 85 张 (8500 元)
- 珂玛转债：1.72 元/股 → 1000 股可配 17 张 (1700 元)
- **相差 5 倍！不能用固定规则计算！**

---

## 基本概念

### 关键日期
- **股权登记日 (Record Date)**: 持有股票可获得配债额度的最后日期
- **发行日 (Issue Date)**: 可转债正式发行日期
- **上市日 (Listing Date)**: 可转债在交易所上市交易日期

## 正确计算公式

```python
# 配债额度 = 持股数 × 每股配售额
配债额度 (元) = 持股数 × 每股配售额

# 可配张数 = floor(配债额度 / 100)
可配张数 = int(配债额度 / 100)

# 所需资金 = 可配张数 × 100
所需资金 (元) = 可配张数 × 100
```

### 交易规则

#### T 日买入股票
- A 股实行 T+1 交收制度
- T 日买入，T+1 日到账
- 必须在股权登记日 **前 1 个交易日** 买入并持有

#### 配债操作
1. 股权登记日前持有股票
2. 发行日在交易软件中操作"配债"
3. 输入配债代码和数量
4. 确保账户有足够资金

### 注意事项

1. **时间窗口**: 从公告到登记日通常只有 3-5 个交易日
2. **抢权风险**: 为配债买入股票可能面临股价下跌风险
3. **破发风险**: 新债上市可能跌破发行价 (100 元)
4. **资金准备**: 配债需要冻结资金，确保账户余额充足
