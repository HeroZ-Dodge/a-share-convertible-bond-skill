## 🚀 使用方式

### 潜伏策略监控

```bash
# 日常监控（自动保存 + 自动进化）
python scripts/monitor/monitor_latent_strategy.py --once

# 查看进化报告
python scripts/monitor/monitor_latent_strategy.py --report

# 导出数据
python scripts/monitor/monitor_latent_strategy.py --export
```
