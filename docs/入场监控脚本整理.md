## 🚀 使用方式

### 日常监控

```bash
# 入场时机监控（每日使用）
python scripts/monitor/monitor_entry_signals.py --once --min-rating B

# 潜伏策略监控（每日使用）
python scripts/monitor/monitor_latent_strategy.py --once
```

### 回测验证

```bash
# 潜伏策略回测
python scripts/monitor/backtest_latent_strategy.py

# 正确策略回测
python scripts/monitor/backtest_correct_strategy.py
```

### 查看说明

```bash
# 查看监控脚本使用说明
cat scripts/monitor/README.md
```
