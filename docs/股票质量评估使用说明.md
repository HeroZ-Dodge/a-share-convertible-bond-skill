# 股票质量评估使用说明

**更新时间**: 2026-04-22  
**功能**: 优化可转债抢权配债策略的选股逻辑

---

## 🎯 功能概述

在原有时间策略基础上，新增**股票质量评估**功能，自动筛选优质正股，提高策略胜率。

### 评估维度

| 维度 | 分值 | 评估指标 |
|------|------|----------|
| **趋势评分** | 40 分 | 均线排列、价格趋势 |
| **动量评分** | 30 分 | 5 日/10 日/20 日涨跌幅 |
| **成交量评分** | 20 分 | 量价配合、成交量趋势 |
| **波动性评分** | 10 分 | 价格稳定性 |
| **总计** | 100 分 | - |

### 评级标准

| 评级 | 分数范围 | 推荐操作 |
|------|----------|----------|
| **A 级** | 80-100 分 | 🟢 优质股，强烈推荐参与 |
| **B 级** | 60-79 分 | 🟢 良好股，推荐参与 |
| **C 级** | 40-59 分 | 🟡 一般股，谨慎参与 |
| **D 级** | 0-39 分 | 🔴 差股，建议回避 |

---

## 🚀 使用方法

### 1. 运行监控脚本（带质量评估）

```bash
# 默认模式：筛选 B 级及以上股票
python3 monitor_entry_signals.py --once

# 只筛选 A 级优质股（更严格）
python3 monitor_entry_signals.py --once --min-rating A

# 放宽标准：C 级及以上即可
python3 monitor_entry_signals.py --once --min-rating C

# 禁用质量评估（使用原始逻辑）
python3 monitor_entry_signals.py --once --no-quality
```

### 2. 持续监控模式

```bash
# 每 60 分钟检查一次，筛选 B 级及以上
python3 monitor_entry_signals.py --interval 60

# 每 30 分钟检查一次，只选 A 级股
python3 monitor_entry_signals.py --interval 30 --min-rating A
```

### 3. 在代码中使用

```python
from lib.stock_quality import StockQualityEvaluator
from lib.data_source import SinaFinanceAPI

# 初始化
sina = SinaFinanceAPI()
evaluator = StockQualityEvaluator(sina_api=sina)

# 评估单只股票
quality = evaluator.evaluate('300622')

print(f"总分：{quality['total_score']:.1f}/100")
print(f"评级：{quality['rating']}")
print(f"推荐：{quality['recommendation']}")

# 详细评分
print(f"趋势：{quality['trend_score']:.1f}/40")
print(f"动量：{quality['momentum_score']:.1f}/30")
print(f"成交量：{quality['volume_score']:.1f}/20")
print(f"波动性：{quality['volatility_score']:.1f}/10")
```

## 🔧 参数说明

### 命令行参数

| 参数 | 说明 | 默认值 | 选项 |
|------|------|--------|------|
| `--once` | 只运行一次 | - | - |
| `--interval` | 监控间隔（分钟） | 60 | - |
| `--no-quality` | 禁用股票质量评估 | False | - |
| `--min-rating` | 最低股票质量评级 | B | A/B/C |

### 评级选择建议

| 场景 | 推荐参数 | 说明 |
|------|----------|------|
| **保守策略** | `--min-rating A` | 只参与优质股，胜率最高 |
| **平衡策略** | `--min-rating B` | 平衡机会数量和质量 |
| **激进策略** | `--min-rating C` | 参与更多机会，风险较高 |
| **回测对比** | `--no-quality` | 使用原始逻辑进行对比 |

## 📈 评估逻辑详解

### 1. 趋势评分 (40 分)

**均线排列 (20 分)**:
- 20 分：完美多头排列 (MA5 > MA10 > MA20 > MA60)
- 15 分：短期多头排列 (MA5 > MA10 > MA20)
- 10 分：超短期多头 (MA5 > MA10)
- 5 分：弱多头 (MA10 > MA20)
- 0 分：空头排列或震荡

**价格趋势 (20 分)**:
- 10 分：当前价 > 60 日均线
- 7 分：当前价 > 20 日均线
- 3 分：当前价 > 10 日均线

### 2. 动量评分 (30 分)

**5 日涨跌幅 (10 分)**:
- 10 分：涨幅 > 5%
- 8 分：涨幅 > 2%
- 5 分：涨幅 > 0%
- 3 分：跌幅 < 5%
- 0 分：跌幅 ≥ 5%

**10 日涨跌幅 (10 分)**:
- 10 分：涨幅 > 10%
- 8 分：涨幅 > 5%
- 5 分：涨幅 > 0%
- 3 分：跌幅 < 5%
- 0 分：跌幅 ≥ 5%

**20 日涨跌幅 (10 分)**:
- 10 分：涨幅 > 15%
- 8 分：涨幅 > 8%
- 5 分：涨幅 > 0%
- 3 分：跌幅 < 10%
- 0 分：跌幅 ≥ 10%

### 3. 成交量评分 (20 分)

**量价配合 (10 分)**:
- 上涨放量 + 下跌缩量 = 高分
- 上涨缩量 + 下跌放量 = 低分

**成交量趋势 (10 分)**:
- 10 分：近期明显放量 (>20%)
- 7 分：温和放量
- 5 分：成交量持平
- 3 分：成交量萎缩

### 4. 波动性评分 (10 分)

**年化波动率**:
- 5 分：基础分
- +5 分：15%-40% (适中)
- +3 分：10%-15% 或 40%-60% (偏低/偏高)
- +1 分：<10% (过低)
- +0 分：>60% (过高)

## ⚠️ 注意事项

1. **数据依赖**: 需要至少 20 个交易日的股价数据才能准确评估
2. **技术面为主**: 当前版本仅评估技术面，未包含基本面指标
3. **动态评估**: 股票质量会随市场变化而动态变化
4. **辅助工具**: 质量评估是辅助工具，不保证 100% 准确
