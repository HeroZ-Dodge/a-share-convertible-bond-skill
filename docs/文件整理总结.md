## 📝 文件分类规则

### lib/ (核心库)
**标准**: 可被其他脚本复用的模块
- 数据源接口
- 计算逻辑
- 工具函数

### scripts/ (脚本)
**标准**: 可独立运行的分析/监控/测试脚本
- 监控脚本：日常运行，监控市场
- 分析脚本：按需运行，分析数据
- 回测脚本：策略验证
- 测试脚本：开发调试

### docs/ (文档)
**标准**: 分析报告、策略说明
- 历史数据分析报告
- 策略回测报告
- 策略说明文档

### reference/ (参考资料)
**标准**: 规则说明、API 文档
- 配债规则
- API 文档
- 决策逻辑
