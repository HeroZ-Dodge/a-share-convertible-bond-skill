# 潜伏策略监控 - 自我进化功能总结

**完成时间**: 2026-04-22  
**版本**: 2.0 🧬

---

## ✅ 完成的功能

### 1. 本地数据存储系统

**模块**: `lib/local_database.py`

**功能**:
- ✅ 自动保存集思录接口获取的待发转债数据
- ✅ 保存历史监控信号
- ✅ 跟踪信号结果（盈亏、持有时间等）
- ✅ 统计进化数据
- ✅ 数据导出功能

**数据文件**:
```
data/
├── pending_bonds_history.json    # 待发转债历史 (每次获取都保存)
├── signals_history.json          # 监控信号历史
├── outcomes_history.json         # 信号结果跟踪
├── evolution_stats.json          # 进化统计
└── evolved_params.json           # 进化后的参数
```

---

### 2. 自我进化引擎

**模块**: `lib/self_evolution.py`

**功能**:
- ✅ 分析历史信号表现
- ✅ 按质量评级分组统计
- ✅ 按时间窗口分组统计
- ✅ 按信号强度分组统计
- ✅ 自动优化监控参数
- ✅ 生成进化报告
- ✅ 提供进化建议

**进化逻辑**:
1. 收集历史信号和结果
2. 分析不同参数组合的表现
3. 找出最优参数（成功率>70% 的配置）
4. 自动更新监控参数
5. 下次监控使用优化后的参数

---

### 3. 监控脚本升级

**脚本**: `scripts/monitor/monitor_latent_strategy.py`

**新增功能**:
- ✅ 自动保存集思录数据
- ✅ 使用进化后的参数进行监控
- ✅ 显示进化状态和建议
- ✅ 自动执行进化（每次监控后）
- ✅ 支持导出进化报告
- ✅ 支持导出数据

**新增参数**:
```bash
--no-save     # 不保存数据
--no-evolve   # 禁用自动进化
--export      # 导出数据
--report      # 显示进化报告
```

## 🧬 自我进化流程

```
第 1 次监控
  ↓
保存数据 (集思录数据 + 信号)
  ↓
用户手动跟踪结果 (可选)
  ↓
第 2 次监控
  ↓
使用初始参数监控
  ↓
保存数据 + 更新统计
  ↓
... 积累 10+ 个案例 ...
  ↓
自动进化触发
  ↓
分析历史数据
  ├─ 按质量评级统计 → 优化 min_quality_rating
  ├─ 按时间窗口统计 → 优化 min/max_days
  └─ 按信号强度统计 → 优化信号阈值
  ↓
保存进化后的参数
  ↓
后续监控使用优化参数
```

### 成熟阶段 (30+ 个案例)
- 参数稳定在最优配置
- 成功率稳定在 75%+
- 持续微调优化

## 🔧 手动跟踪结果

为了让进化更准确，建议手动跟踪信号结果：

**示例脚本** `scripts/monitor/add_outcome.py`:
```python
#!/usr/bin/env python3
import sys, os
sys.path.insert(0, os.path.dirname(os.path.dirname(os.path.abspath(__file__))))

from lib.local_database import LocalDatabase

db = LocalDatabase()

db.save_outcome({
    'bond_code': '123456',
    'bond_name': 'XX 转债',
    'signal_date': '2026-04-22',
    'signal_type': 'latent',
    'entry_price': 45.80,
    'exit_price': 48.50,
    'exit_date': '2026-04-25',
    'return': 5.9,
    'success': True,
    'hold_days': 3,
    'stock_quality': {'rating': 'B', 'total_score': 75},
    'days_since_tongguo': 38,
})

print('✅ 结果已记录!')
```

## 🎯 核心优势

1. **数据沉淀**: 所有历史数据本地保存，不会丢失
2. **持续学习**: 每次监控都在积累经验
3. **自动优化**: 无需人工调整参数
