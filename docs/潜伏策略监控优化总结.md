### 1. 信号检测条件优化

**问题**: 原始条件太严格，实际市场环境下很难触发信号

**优化前**:
```python
成交量 > 2.0x (阈值)
2 日涨跌 > 2.0%
5 日涨跌 > 3.0%
需要 ≥2 个信号
```

**优化后**:
```python
成交量 > 1.5x (降低到 1.5 倍)
2 日涨跌 > 1.5% (降低到 1.5%)
5 日涨跌 > 2.5% (降低到 2.5%)
接近 20 日高点 (>95%) 也算信号
需要 ≥2 个信号，或 1 个强信号 + 成交量 >2.0x
```

**效果**: 成功检测到 1 个信号 (300881 盛德鑫泰)

## 🔧 技术细节

### 信号检测逻辑

```python
# 1. 成交量信号
if vol_ratio > 1.5:
    signal_count += 1
    signal_details.append(f'成交量{vol_ratio:.1f}倍')

# 2. 2 日涨跌幅信号
if price_change_2d > 1.5:
    signal_count += 1
    signal_details.append(f'2 日{price_change_2d:+.1f}%')

# 3. 5 日涨跌幅信号
if price_change_5d > 2.5:
    signal_count += 1
    signal_details.append(f'5 日{price_change_5d:+.1f}%')

# 4. 突破 20 日高点
if latest_close > high_20d:
    signal_count += 1
    signal_details.append('突破 20 日高点')

# 5. 接近高点 (新增)
if latest_close > high_20d * 0.95:
    signal_details.append(f'接近 20 日高点 ({latest_close/high_20d*100:.1f}%)')

# 6. 信号判断
is_strong_signal = signal_count >= 2 or (signal_count >= 1 and vol_ratio > 2.0)
```
