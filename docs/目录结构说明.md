## 📚 脚本分类说明

### 监控脚本 (日常使用)

| 脚本 | 用途 | 使用频率 |
|------|------|----------|
| `monitor_entry_signals.py` | 监控入场时机，自动筛选优质股 | 每日 |
| `monitor_latent_strategy.py` | 潜伏策略监控，捕捉消息泄露信号 | 每日 |

**使用示例**:
```bash
# 入场时机监控
python scripts/monitor_entry_signals.py --once --min-rating B

# 潜伏策略监控
python scripts/monitor_latent_strategy.py --once
```

---

### 分析脚本 (按需使用)

| 脚本 | 用途 |
|------|------|
| `analyze_pending.py` | 查看待发转债列表 |
| `analyze_quequan_profit.py` | 分析历史配债收益 |
| `analyze_compact.py` | 输出紧凑摘要 |

**使用示例**:
```bash
# 查看待发转债
python scripts/analyze_pending.py --compact

# 分析历史收益
python scripts/analyze_quequan_profit.py --limit 10
```

## 🔧 核心模块 (lib)

### data_source.py
统一数据源接口：
- 集思录 API (待发转债)
- 东方财富 API (转债列表)
- 新浪财经 API (股票历史价格)

### bond_calculator.py
配债计算：
- 配债额度计算
- 持股数计算
- 盈亏分析

### stock_quality.py 🆕
股票质量评估：
- 趋势评分 (40 分)
- 动量评分 (30 分)
- 成交量评分 (20 分)
- 波动性评分 (10 分)

### report.py
报告生成工具

## 📝 快速开始

### 日常监控
```bash
# 每天运行一次，监控入场机会
python scripts/monitor_entry_signals.py --once --min-rating B
```

### 潜伏策略
```bash
# 监控消息泄露信号
python scripts/monitor_latent_strategy.py --once
```

### 查看待发转债
```bash
# 紧凑模式
python scripts/analyze_pending.py --compact
```

---

## 🎯 推荐工作流

1. **每日**: 运行 `monitor_entry_signals.py` 检查入场机会
2. **每日**: 运行 `monitor_latent_strategy.py` 检查潜伏信号
3. **每周**: 运行 `analyze_pending.py` 查看新转债
4. **按需**: 运行回测脚本验证策略

---

**提示**: 所有脚本都支持 `--help` 查看参数说明。
