## 🚀 快速开始


## 📊 策略说明

### 核心逻辑

统计显示：**同意注册前 5 天平均涨幅 +4.56%，上涨概率 81.8%**

这说明有资金可能提前获知消息并布局。本脚本监控以下信号：

1. **时间窗口**: 上市委通过后 25-55 天（自动进化优化）
2. **股价上涨**: 连续 2 日上涨 >2%
3. **突破高点**: 股价突破 20 日高点
4. **股票质量**: B 级及以上（自动进化优化）

### 预期效果

| 指标 | 预期值 |
|------|--------|
| 胜率 | ~70-75% |
| 平均收益 | +4~6% |
| 持有时间 | 5-25 天 |
| 年化收益 | ~100% |

## 🔧 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--once` | 只运行一次 | - |
| `--interval` | 监控间隔（分钟） | 60 |
| `--no-save` | 不保存数据到本地数据库 | False |
| `--no-evolve` | 禁用自动进化 | False |
| `--export` | 导出数据 | False |
| `--report` | 显示进化报告 | False |

## 🧬 自我进化功能

### 进化原理

1. **数据收集**: 自动保存集思录数据和监控信号
2. **结果跟踪**: 记录信号后续盈亏（需手动添加）
3. **统计分析**: 按质量/时间窗口/信号强度分组统计
4. **参数优化**: 自动找出最优参数配置
5. **持续学习**: 每次监控都在积累经验

## ⚠️ 注意事项

1. **数据积累**: 前 10 个案例进化效果不明显，需要耐心
2. **结果跟踪**: 建议手动跟踪信号结果，提升进化准确性
3. **定期备份**: 建议定期备份 `data/` 目录
4. **止损位**: 建议设置 -5% 止损
5. **仓位控制**: 单只标的仓位不超过 20%
