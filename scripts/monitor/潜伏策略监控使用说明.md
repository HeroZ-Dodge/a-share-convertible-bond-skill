## 🧬 自我进化原理

### 1. 数据收集阶段
每次监控自动保存:
- 集思录待发转债数据
- 检测到的信号
- 信号参数（质量评级、时间窗口、信号强度）

### 2. 结果跟踪阶段
手动或自动更新信号结果:
```python
from lib.local_database import LocalDatabase

db = LocalDatabase()
db.save_outcome({
    'bond_code': '123456',
    'signal_date': '2026-04-22',
    'signal_type': 'latent',
    'entry_price': 45.80,
    'exit_price': 48.50,
    'exit_date': '2026-04-25',
    'return': 5.9,
    'success': True,
    'hold_days': 3,
    'stock_quality': {'rating': 'B', 'score': 75},
    'days_since_tongguo': 38,
})
```

### 3. 参数优化阶段
基于历史数据自动调整:
- **时间窗口**: 找出成功率最高的时间段
- **质量评级**: 调整最低评级要求
- **信号强度**: 优化信号数量阈值

### 4. 持续学习
每次监控都会:
1. 使用进化后的参数
2. 收集新的案例
3. 更新统计和优化参数

## 🔧 高级用法

### 手动添加结果跟踪

创建脚本 `scripts/monitor/add_outcome.py`:

```python
#!/usr/bin/env python3
import sys
import os
sys.path.insert(0, os.path.dirname(os.path.dirname(os.path.abspath(__file__))))

from lib.local_database import LocalDatabase

db = LocalDatabase()

# 添加结果记录
db.save_outcome({
    'bond_code': '123456',
    'bond_name': 'XX 转债',
    'signal_date': '2026-04-22',
    'signal_type': 'latent',
    'entry_price': 45.80,
    'exit_price': 48.50,
    'exit_date': '2026-04-25',
    'return': 5.9,
    'success': True,
    'hold_days': 3,
    'stock_quality': {'rating': 'B', 'total_score': 75},
    'days_since_tongguo': 38,
    'notes': '符合预期，顺利止盈',
})

print('结果已记录!')
```

## 📈 数据可视化 (可选)

使用 Jupyter Notebook 分析数据:

```python
import json
import pandas as pd
import matplotlib.pyplot as plt

# 加载数据
with open('data/outcomes_history.json') as f:
    outcomes = json.load(f)['outcomes']

# 转换为 DataFrame
df = pd.DataFrame(outcomes)

# 绘制成功率趋势
plt.figure(figsize=(10, 6))
df['success'].cumsum() / range(1, len(df)+1) * 100
plt.plot(df['success'].cumsum() / range(1, len(df)+1) * 100)
plt.xlabel('信号数量')
plt.ylabel('累计成功率 (%)')
plt.title('潜伏策略成功率进化趋势')
plt.grid(True)
plt.show()
```
