## 🔍 核心发现：股价三阶段规律

通过分析注册前 30 天的股价运行规律，发现明显的**三阶段特征**：

### 阶段 1：建仓期（注册前 20-30 天）
- **胜率**: 0-17%
- **平均收益**: -8.8% 到 -12.2%
- **特征**: 资金试探性建仓，股价上涨但持有到注册日反而亏损
- **结论**: ❌ **不要入场**

### 阶段 2：洗盘期（注册前 7-15 天）
- **胜率**: 0-33%
- **平均收益**: -2.4% 到 -11%
- **特征**: 主力洗盘，股价回调 10-15%，成交量萎缩
- **结论**: ⚠️ **谨慎观望**

### 阶段 3：拉升期（注册前 0-7 天）
- **胜率**: 67-83%
- **平均收益**: +1.3% 到 +3.4%
- **特征**: 洗盘结束，股价重新拉升，2 日涨幅转正
- **结论**: ✅ **最佳入场时机**

## 📈 新信号策略设计

基于分析，设计**洗盘结束信号**：

### 信号条件

```python
def check_washout_end_signal(prices, stock_code):
    """
    检测洗盘结束信号
    
    条件：
    1. 从近期高点回撤 > 10%
    2. 量比 < 1.0（缩量）
    3. 2 日涨幅 > 0（开始企稳）
    4. 5 日跌幅 > -15%（回调未过度）
    """
    # 计算从近期高点的回撤
    recent_high = max(prices[-20:]['high'])
    current_close = prices[-1]['close']
    drawdown = (current_close - recent_high) / recent_high * 100
    
    # 计算量比
    current_vol = prices[-1]['volume']
    avg_vol_10 = prices[-10:]['volume'].mean()
    vol_ratio = current_vol / avg_vol_10
    
    # 计算 2 日涨幅
    close_2d = prices[-3]['close']
    change_2d = (current_close - close_2d) / close_2d * 100
    
    # 计算 5 日跌幅
    close_5d = prices[-6]['close']
    change_5d = (current_close - close_5d) / close_5d * 100
    
    # 信号判断
    has_signal = (
        drawdown < -10 and  # 回撤超过 10%
        vol_ratio < 1.0 and  # 缩量
        change_2d > 0 and  # 2 日转正
        change_5d > -15  # 5 日跌幅未超过 15%
    )
    
    return {
        'has_signal': has_signal,
        'drawdown': drawdown,
        'vol_ratio': vol_ratio,
        'change_2d': change_2d,
        'change_5d': change_5d,
    }
```

## 💡 结论与建议

### 核心规律

1. **注册前 1-7 天**是最佳入场窗口（胜率 83%，收益 +1.3%~+3.4%）
2. **注册前 20-30 天**的信号几乎都亏钱（胜率 0-17%，收益 -8.8%~-12.2%）
3. **洗盘结束的特征**：回撤 10-20% + 缩量 + 2 日涨幅转正

### 实操建议

#### 策略：洗盘结束信号入场

**入场条件**：
1. 上市委通过后 15 天开始监控
2. 从近期高点回撤 > 10%
3. 量比 < 1.0（缩量）
4. 2 日涨幅 > 0（企稳信号）
5. 5 日跌幅 > -15%（回调未过度）

**出场条件**：
- 同意注册公告发布后卖出
- 或持有 10 个交易日后卖出

**风控**：
- 止损：-5%
- 单只仓位：不超过 20%

#### 为什么这个策略可行？

1. **不需要预知注册日**：只关注股价和成交量信号
2. **洗盘结束是可见的**：缩量 + 企稳是明确的拐点信号
3. **胜率较高**：历史数据显示注册前 1-7 天入场胜率 83%

### 待优化方向

1. **扩大样本量**：当前仅 6 只，需要更多历史数据验证
2. **加入行业因子**：不同行业可能有不同规律
3. **加入大盘环境**：牛市/熊市策略表现可能不同
4. **动态参数**：根据市场情况调整回撤阈值
